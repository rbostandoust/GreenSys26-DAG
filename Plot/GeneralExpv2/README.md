# Heterogen Folder
- Objective: $min(CarbonConsumption)$
- Results for running on a heterogeneous settings
- Has Line plots for Average savings and Box plots

# Heterogen_Energy Folder
- Objective: $min(EnergyConsumption + CarbonTieBreaker)$
- Results for running on a heterogeneous settings
- Has Line plots for Average savings

# Homogen Folder
- Objective: $min(CarbonConsumption)$
- Results for running on a homogeneous settings
- Has Line plots for Average savings and Box plots

# File Naming Format
- {#Jobs}J_{#Servers}S_{#OPS}O_{PlotFormat}_{Location}.[csv, png, pdf]
    - PlotFormat = [BoxPlot or LinePlot]
